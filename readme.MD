# 📝 Task Manager - Flask CRUD App

A simple task manager built with Flask and deployed on Heroku.

## Features
- Add tasks
- View all tasks
- Update tasks
- Delete tasks

## Tech Stack
- Python
- Flask
- SQLite (via SQLAlchemy)
- Deployed on Heroku

## Run locally
```bash
pip install -r requirements.txt
python app.py
```
